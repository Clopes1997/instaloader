Welcome to Instaloader

A free, open browser extension that lets you browse any website and download every image or video you see — in one click.

⚗️
AI-assisted development ahead — this extension was built using a combination of ChatGPT Codex, Cursor, and Claude Code. It handles static images and direct video files well, but it cannot download adaptive streams such as MediaSource / HLS / DASH (used by platforms like YouTube, Twitch, Netflix, and many modern video sites). For those cases, consider using yt-dlp or a browser screen recorder.

What it does

🔍

Finds everything

Detects images and videos from <img>, <picture>, CSS backgrounds, and <video> elements.

📦

ZIP download

Download all visible images in a single ZIP file — no 50 save dialogs.

🎬

Video support

Detects MP4, WebM, MOV and other direct video URLs playing in the page.

🔎

Smart filters

Filter by type (images / videos) and size (large 500 px+, medium, any).

🛡️

Private by design

No servers, no tracking. Everything runs locally in your browser.

⚡

Works anywhere

News sites, portfolios, social feeds, e-commerce — any page you can visit.

How to use

1

Navigate to any page

Open any website that has images or videos you want to save.

2

Click the Instaloader icon

Click the 🖼️ icon in your browser toolbar. The extension scans the page and shows every image and video it finds as a thumbnail grid.

3

Filter what you want

Use the Images / Videos tabs to narrow by type, and Large 500+ / Medium to filter by size. Set a custom minimum pixel size in the input field.

4

Download

Click a thumbnail to download that single file. Download Visible as ZIP bundles all visible images into one ZIP file (videos are downloaded individually by Chrome's download manager).

5

Copy a URL

Right-click any thumbnail to copy the direct media URL to your clipboard.
